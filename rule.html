ルール

参加要件

所属によらず，どなたでも参加することが可能です． AtCoder 上でコンテストを行うので，事前に AtCoder 上のアカウントを取得しておいて下さい．

環境

AtCoder で使用可能な言語は全て使用可能にする予定です．

順位付け

  • 問題にはそれぞれ点数が割り当てられており，1 つの問題を解くごとにその問題に対応する点数を獲得します．問題によっては部分点が設けられており，解答が一定の条件を満たすと対応する点数を獲得します．
  • 順位は獲得した得点が高いほど上になります．得点が同じ場合，ペナルティタイムが小さいほど上になります．
  • ペナルティタイムは，（最後に正答と判定された解答を提出した時間） + （正答した問題に対する誤答の数の合計 ×5分）です．

注意事項

  • 自作ライブラリの使用，インターネットでの検索は許可されます．
  • 提出されたソースコードはコンテスト終了後に公開されます．
  • コンテストは個人戦とします．2人以上で結託し解答する行為は禁止しております．
  • プログラムの実行制限時間・使用メモリ制限は各テストケースごとに課されます．
  • Compilation Error はペナルティタイムには加算されません．
  • コンテスト中何か質問等がある場合は AtCoder 上から質問できます．
  • 問題文の変更や再ジャッジが発生した場合，AtCoder 上でアナウンスされます．
  • Javaをお使いの場合，クラス名は「Main」にして下さい．
  • C をお使いの場合，main関数が必ず0を返すようにして下さい．(return 0; を書いておけばよいです)
  • 順位表の凍結はありません．

禁止事項

以下の行為を固く禁じます．
  • チームでの参加．(当コンテストは個人戦とします．)
  • コンテスト時間中にコンテスト中に解法などをウェブサイト上へ書き込む行為．
  • システムに対し攻撃をかける行為．
